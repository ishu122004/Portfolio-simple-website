# Personal Portfolio Website

This is a personal portfolio website developed using HTML and basic CSS to showcase my profile, skills, projects, and contact information. The website consists of five pages: Home, About, Skills, Projects, and Contact, providing a structured overview of my background and technical abilities.

## Features
- Home page with personal introduction
- About page describing my background and career interests
- Skills page highlighting technical competencies
- Projects page showcasing academic and personal projects
- Contact page with communication details
- Simple, clean, and user-friendly design

## Technologies Used
- HTML5
- CSS3

## Purpose
This portfolio website was created to present my skills, achievements, and projects in a professional manner while demonstrating my understanding of web development fundamentals using HTML and CSS.

Thank you for visiting my portfolio website.
